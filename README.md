# Project-Test2-Nadya: Chatbot Demo (Mock AI)

## Deskripsi
Project ini adalah chatbot demo menggunakan **Mock AI**, meniru alur AI Chat GPT/Gemini:
- User bisa kirim pesan
- AI membalas secara otomatis
- Integrasi frontend (EJS) + backend (Express) + endpoint chat `/chat`

> Catatan: AI di sini **mock**, bukan API beneran. Tujuan demo:
> - Menunjukkan integrasi frontend → backend → AI
> - Flow siap untuk diganti ke OpenAI/Gemini jika API aktif

Teknologi:
- Node.js + Express
- EJS
- JavaScript + fetch API
- CSS sederhana

---

## Fitur
- Form chat sederhana
- Chat log menampilkan pesan user & balasan AI
- Balasan AI random / echo supaya terlihat hidup
